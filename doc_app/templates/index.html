About Numerical Methods

Numerical computing is an approach of solving complex mathematical problems which can not be solved easily by analytical mathematics by using simple arithmetic operations and which requires development, analysis and use of an algorithm along with some computing tools.

In numerical analysis mathematical problems are formulated and solved with arithmetic operations and these arithmetic operations are carried out with the help of high level programming languages like C, C++, Python, Matlab etc on computer.
